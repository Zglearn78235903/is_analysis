#评定成绩用例

###1.用例规约

| 用例名称  |  评定成绩 |
| :-----| ----: | 
| 功能 | 老师评定一个学生的实验成绩 |
| 参与者	 | 老师 |
| 条件 | 需要先登录 |
| 主事件流	 | 输入一个或者多个实验的成绩和评语 |
| 备选事件流 | 成绩必须是0至100之间，可以为空，但不能超界|
