#登录用例

###1.用例规约

| 用例名称  |  登录 |
| :-----| ----: | 
| 功能 | 登录 |
| 参与者	 | 用户 |
| 条件 | 登录成功后，跳转到主页 |
| 主事件流	 | 1. 访客输入用户名和密码，选择用户类型2.系统判断用户名，密码，用户类正确，允许登录 |
| 备选事件流 | 1.输入的用户名或者密码为空 2.系统判断用户名，密码，用户类不正确，不允许登录|
